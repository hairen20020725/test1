# 管理后台访问说明

## 访问方式

管理后台已从用户界面中移除，不会在普通用户界面显示任何入口。

### 访问地址

直接在浏览器中访问：`/admin/login`

例如：
- 本地开发：`http://localhost:5173/admin/login`
- 生产环境：`https://your-domain.com/admin/login`

### 登录凭证

**默认密码：** `admin123`

### 修改密码

可以通过环境变量配置管理员密码：

1. 在项目根目录的 `.env` 文件中添加：
   ```
   VITE_ADMIN_PASSWORD=your_custom_password
   ```

2. 重启应用后生效

### 会话管理

- 登录后会话有效期：**2小时**
- 超时后需要重新登录
- 可以在任何管理页面点击"退出登录"按钮主动退出

### 管理功能

登录后可以访问以下功能：

1. **产品管理** (`/admin/products`)
   - 添加、编辑、删除空调产品
   - 管理库存和价格信息

2. **案例管理** (`/admin/cases`)
   - 添加、编辑、删除历史案例
   - 上传户型图
   - 记录解决方案

### 安全说明

- 所有管理页面都受密码保护
- 未登录用户访问管理页面会自动跳转到登录页
- 会话信息存储在浏览器的 sessionStorage 中
- 关闭浏览器后会话自动清除

### 注意事项

- 请妥善保管管理员密码
- 建议在生产环境中修改默认密码
- 不要在公共场合分享管理后台访问地址
