# 管理后台使用指南

## 📋 概述

智能空调方案推荐系统现已集成完整的管理后台，您可以通过管理后台添加和管理：
- ✅ **产品库存**：空调产品的详细信息、价格、库存
- ✅ **历史案例**：成功的空调安装案例，包括户型图和解决方案

## 🚀 快速开始

### 访问管理后台

1. 打开应用首页
2. 点击页面顶部的 **"管理后台"** 按钮
3. 进入管理后台首页，选择要管理的模块

### 访问路径

- 首页：`/`
- 管理后台首页：`/admin`
- 产品管理：`/admin/products`
- 案例管理：`/admin/cases`

## 📦 产品管理

### 添加产品

1. 进入 **产品管理** 页面
2. 点击右上角 **"添加产品"** 按钮
3. 填写产品信息：
   - **基本信息**：产品ID、品牌、型号、类型
   - **规格参数**：匹数、适用面积、能效等级
   - **价格库存**：当前售价、原价、库存数量
   - **性能参数**：噪音、制冷量、制热量
   - **描述信息**：产品特点、适用场景
   - **促销信息**：可选，填写促销活动
4. 点击 **"添加"** 保存

### 编辑产品

1. 在产品列表中找到要编辑的产品
2. 点击 **"编辑"** 按钮
3. 修改需要更新的信息
4. 点击 **"更新"** 保存

### 删除产品

1. 在产品列表中找到要删除的产品
2. 点击 **删除图标** 按钮
3. 确认删除操作

### 产品字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 产品ID | 唯一标识符，不可重复 | `split-007` |
| 类型 | 中央空调/分体式/风管机/移动空调 | `分体式` |
| 品牌 | 产品品牌 | `格力` |
| 型号 | 产品型号 | `KFR-35GW` |
| 匹数 | 空调功率，支持小数 | `1.5` |
| 适用面积 | 最小和最大面积（平方米） | `15-22` |
| 能效等级 | 能效标准 | `一级能效` |
| 当前售价 | 实际销售价格（元） | `3100` |
| 原价 | 可选，用于显示优惠 | `3500` |
| 库存数量 | 当前库存台数 | `25` |
| 噪音 | 运行噪音（分贝） | `38` |
| 制冷量 | 制冷功率（瓦特） | `3500` |
| 制热量 | 制热功率（瓦特） | `4200` |
| 产品特点 | 用"、"分隔 | `静音舒适、快速制冷、智能控制` |
| 适用场景 | 用"、"分隔 | `卧室、书房、小客厅` |
| 促销信息 | 可选 | `热销款，优惠400元` |

## 📝 案例管理

### 查看案例

1. 进入 **案例管理** 页面
2. 浏览所有历史案例
3. 查看案例详情：
   - 户型信息（面积、房间、朝向等）
   - 解决方案（产品配置、费用）
   - 客户反馈

### 添加案例

1. 进入 **案例管理** 页面
2. 点击右上角 **"添加案例"** 按钮
3. 填写案例信息：

   **基本信息**：
   - 案例ID：唯一标识符（如：case-006）
   - 案例标题：简洁描述（如：阳光花园120㎡三室两厅中央空调方案）
   - 完成日期：案例完成时间

   **户型信息**：
   - 面积：房屋总面积（平方米）
   - 户型：房间配置（如：3室2厅）
   - 朝向：房屋朝向（南向、北向等）
   - 楼层：所在楼层
   - 建筑类型：高层、多层、别墅等
   - 户型描述：详细描述户型特点

   **户型图上传**：
   - 支持格式：JPG、PNG、WEBP
   - 文件大小：最大1MB
   - 文件名：不能包含中文字符
   - 点击上传区域选择图片

   **解决方案**：
   - 方案类型：中央空调、风管机+分体式等
   - 产品配置：
     - 点击"添加产品"添加多个产品
     - 每个产品包含：房间、产品、数量、安装位置
     - 可删除不需要的产品
   - 设备费用：所有设备的总费用
   - 安装费用：安装和辅材费用

   **客户反馈与经验**：
   - 客户反馈：记录客户的评价和使用体验
   - 注意事项：每行一条经验，自动转换为列表

4. 点击 **"添加案例"** 保存

### 编辑案例

1. 在案例列表中找到要编辑的案例
2. 点击 **"编辑"** 按钮
3. 修改需要更新的信息
4. 如需更换户型图，重新上传即可
5. 点击 **"更新案例"** 保存

### 删除案例

1. 在案例列表中找到要删除的案例
2. 点击 **删除图标** 按钮
3. 确认删除操作

### 案例字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 案例ID | 唯一标识符，不可重复 | `case-006` |
| 标题 | 案例简要描述 | `阳光花园120㎡三室两厅中央空调方案` |
| 面积 | 房屋总面积（㎡） | `120` |
| 户型 | 房间配置 | `3室2厅` |
| 朝向 | 房屋朝向 | `南向` |
| 楼层 | 所在楼层 | `10` |
| 建筑类型 | 建筑形式 | `高层` |
| 户型描述 | 详细特点 | `南北通透，客厅朝南...` |
| 户型图 | 图片文件 | 上传JPG/PNG/WEBP |
| 方案类型 | 空调配置类型 | `中央空调` |
| 产品配置 | 产品列表 | 多个产品组合 |
| 设备费用 | 设备总价（元） | `35000` |
| 安装费用 | 安装费（元） | `5000` |
| 客户反馈 | 客户评价 | `使用效果很好...` |
| 注意事项 | 经验总结 | 每行一条 |

### 户型图上传注意事项

1. **文件格式**：
   - ✅ 支持：JPG、JPEG、PNG、WEBP
   - ❌ 不支持：GIF、BMP、SVG等

2. **文件大小**：
   - 最大1MB
   - 建议压缩后上传
   - 系统会自动验证

3. **文件命名**：
   - ❌ 不能包含中文字符
   - ✅ 使用英文、数字、下划线
   - 示例：`floor_plan_001.jpg`

4. **图片质量**：
   - 建议分辨率：800x600以上
   - 确保户型图清晰可见
   - 避免过度压缩导致模糊

## 🤖 AI推荐机制

### 数据使用

AI推荐系统会自动使用管理后台中的数据：

1. **产品数据**：
   - 优先推荐有库存的产品
   - 考虑产品价格和促销信息
   - 根据适用面积匹配房间

2. **历史案例**：
   - 参考相似户型的成功案例
   - 学习产品配置经验
   - 提供费用参考

### 推荐原则

AI会遵循以下原则生成方案：

1. ✅ **优先推荐有库存的产品** - 确保方案可立即实施
2. ✅ **参考历史案例** - 找到相似户型的成功经验
3. ✅ **面积精准匹配** - 根据房间面积选择合适匹数
4. ✅ **考虑实际预算** - 在预算范围内选择性价比最高的产品
5. ✅ **使用场景优化** - 卧室优先静音，客厅优先制冷量
6. ✅ **户型适配** - 大户型推荐中央空调，小户型推荐分体式
7. ✅ **能效优先** - 优先推荐一级能效产品
8. ✅ **关注促销** - 优先推荐有促销活动的产品

## 💡 最佳实践

### 产品管理建议

1. **保持库存准确**：及时更新库存数量，确保推荐的产品有货
2. **完善产品信息**：详细填写产品特点和适用场景，提高推荐精准度

1. **多样化案例**：添加不同面积、不同户型的案例
2. **详细描述**：记录户型特点、解决方案和客户反馈
3. **上传户型图**：有图片的案例更有说服力（功能开发中）
4. **记录经验**：在"注意事项"中总结安装经验

## ❓ 常见问题

### Q: 修改产品信息后，AI推荐会立即生效吗？

A: 是的，所有修改都会实时同步到数据库，AI推荐会使用最新数据。

### Q: 删除产品会影响历史案例吗？

A: 如果历史案例中引用了被删除的产品，案例仍会保留，但产品信息可能无法显示。建议先检查案例引用再删除产品。

### Q: 如何批量导入产品？

A: 目前暂不支持批量导入，需要逐个添加。如有大量数据需要导入，请联系技术支持。

### Q: 可以上传户型图吗？

A: 案例添加功能（包括户型图上传）正在开发中，敬请期待。
