# 📝 如何添加新的户型图和解决方案

## 🚀 快速开始（3步完成）

### 第1步：进入管理后台

1. 打开应用首页
2. 点击页面顶部的 **"管理后台"** 按钮
3. 或直接访问：`/admin`

### 第2步：进入案例管理

1. 在管理后台首页，点击 **"案例管理"** 卡片
2. 或直接访问：`/admin/cases`
3. 点击右上角的 **"添加案例"** 按钮

### 第3步：填写案例信息

#### 📋 基本信息
```
案例ID：case-006（必须唯一）
案例标题：碧桂园150㎡四室两厅中央空调方案
完成日期：2025-11-27
```

#### 🏠 户型信息
```
面积：150 ㎡
户型：4室2厅
朝向：南向（下拉选择）
楼层：15
建筑类型：高层（下拉选择）
户型描述：
南北通透，客厅朝南采光好，主卧带独立卫生间，
次卧朝南，书房朝北，厨房和餐厅相连。
```

#### 🖼️ 上传户型图
1. 点击上传区域
2. 选择户型图文件
3. **注意**：
   - 支持格式：JPG、PNG、WEBP
   - 文件大小：最大1MB
   - 文件名：不能包含中文（如：floor_plan_006.jpg）

#### 💡 解决方案
```
方案类型：中央空调

产品配置：
┌─────────────────────────────────────┐
│ 产品1                                │
│ 房间：客厅                           │
│ 产品：格力 GMV-H180WL/A（下拉选择）  │
│ 数量：1                              │
│ 安装位置：客厅吊顶中央               │
└─────────────────────────────────────┘

点击"添加产品"可以添加更多产品...

设备费用：45000 元
安装费用：8000 元
```

#### 💬 客户反馈与经验
```
客户反馈：
使用效果非常好，制冷制热速度快，
噪音很小，全屋温度均匀，非常满意。

注意事项（每行一条）：
中央空调需要提前规划吊顶高度，建议预留300mm
出风口位置要避开床头和沙发直吹
定期清洗滤网和检查冷凝水管
建议安装新风系统提升空气质量
```

### 第4步：保存案例

点击页面底部的 **"添加案例"** 按钮

✅ 完成！案例已添加到系统中，AI推荐会自动使用这个案例。

## 📸 户型图准备指南

### 文件要求
- ✅ 格式：JPG、PNG、WEBP
- ✅ 大小：小于1MB
- ✅ 文件名：英文、数字、下划线（如：floor_plan_006.jpg）
- ❌ 不要：中文文件名（如：户型图.jpg）

### 如何压缩图片

#### 方法1：在线压缩（推荐）
1. 访问在线压缩工具（如：tinypng.com）
2. 上传户型图
3. 下载压缩后的文件
4. 重命名为英文文件名

#### 方法2：使用图片编辑软件
1. 打开图片（Windows画图、Mac预览）
2. 调整大小：宽度800-1200像素
3. 保存为JPG格式，质量80%
4. 重命名为英文文件名

### 文件命名示例
```
✅ 正确：
- floor_plan_001.jpg
- case_006_layout.png
- apartment_150sqm.webp

❌ 错误：
- 户型图.jpg
- 案例006.png
- 150平米户型.webp
```

## ❓ 常见问题

### Q1：文件名包含中文怎么办？
**A**：上传前重命名文件，使用英文、数字、下划线。
```
重命名方法：
1. 右键点击文件
2. 选择"重命名"
3. 改为：floor_plan_001.jpg
```

### Q2：图片超过1MB怎么办？
**A**：使用在线压缩工具或图片编辑软件压缩。
```
推荐工具：
- tinypng.com（在线）
- compressjpeg.com（在线）
- Windows画图（本地）
- Mac预览（本地）
```

### Q3：如何选择产品？
**A**：在产品下拉列表中选择，如果没有合适的产品，先去产品管理添加。
```
步骤：
1. 返回管理后台首页
2. 进入产品管理
3. 添加新产品
4. 返回案例管理继续添加案例
```

### Q4：可以添加多个产品吗？
**A**：可以！点击"添加产品"按钮，可以添加任意数量的产品。
```
示例：
- 客厅：中央空调主机
- 主卧：风管机
- 次卧：分体式空调
- 书房：分体式空调
```

### Q5：如何编辑已添加的案例？
**A**：在案例列表中点击"编辑"按钮。
```
步骤：
1. 进入案例管理
2. 找到要编辑的案例
3. 点击"编辑"按钮
4. 修改信息
5. 点击"更新案例"
```
