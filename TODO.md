## 技术栈
- React + TypeScript
- Tailwind CSS + shadcn/ui
- 文心一言多模态AI接口
- ky (HTTP客户端)
- eventsource-parser (SSE流式处理)

## 注意事项
- 不需要Supabase后端（纯前端应用）
- 图片不需要持久化存储，直接转base64发送给AI
- 使用流式接口提升用户体验
- 主色调：天空蓝 #4A90E2
