### 2. 智能推荐机制

AI在分析户型图时，会：

1. **分析房间特征**
   - 识别房间类型（客厅、卧室、书房等）
   - 估算房间面积
   - 考虑朝向影响

2. **匹配产品规格**
   - 根据房间面积选择合适匹数
   - 考虑能效等级和节能性
   - 匹配用户预算范围

3. **推荐具体产品**
   - 从知识库中选择真实产品型号
   - 提供详细的产品信息
   - 说明推荐理由

4. **计算准确预算**
   - 基于真实产品价格
   - 包含设备和安装费用
   - 提供价格区间范围

### 3. 产品展示功能

页面底部的"产品知识库"区域提供：

- **分类浏览**：按产品类型筛选查看
- **详细信息**：每个产品的完整规格
- **对比参考**：方便用户了解不同产品的差异

## 使用流程

### 步骤1：上传户型图
上传清晰的户型图，确保房间结构可见

### 步骤2：填写参数（可选）
- 房间数量：如"3室2厅"
- 主要朝向：选择房屋朝向
- 使用需求：描述特殊要求

### 步骤3：AI分析推荐
AI将：
1. 分析户型结构和面积
2. 从知识库中选择合适产品
3. 为每个房间推荐具体型号
4. 计算总体预算

### 步骤4：查看推荐结果
推荐方案包含：
- 户型分析
- 整体方案建议
- **具体产品推荐**（品牌、型号、价格）
- 预算估算
- 注意事项

### 步骤5：浏览产品知识库
在页面底部查看所有可用产品，了解更多选择

## 产品信息说明

每个产品卡片包含以下信息：

- **基本信息**：品牌、型号、类型
- **能效等级**：一级/新一级/三级能效
- **匹数**：空调功率大小
- **噪音**：运行噪音分贝值
- **制冷量**：制冷能力（W）
- **价格区间**：设备采购价格范围
- **适用面积**：推荐使用面积
- **产品特点**：核心功能和优势
- **适用场景**：最适合的使用场景

## 优势特点

### 1. 真实可靠
- 所有产品数据来自真实市场产品
- 价格区间符合实际市场行情
- 规格参数准确可信

### 2. 精准匹配
- AI根据房间面积精确匹配产品
- 考虑使用场景选择合适型号
- 平衡性能和预算

### 3. 专业推荐
- 遵循空调选型专业原则
- 考虑能效、噪音、制冷量等多维度
- 提供详细的推荐理由

### 4. 预算透明
- 基于真实产品价格计算
- 提供价格区间范围
- 包含设备和安装费用

### 5. 方便对比
- 产品知识库支持分类浏览
- 详细的规格信息便于对比
- 帮助用户做出明智选择

## 注意事项

1. **价格仅供参考**
   - 实际价格可能因地区、促销等因素有所差异
   - 建议以实际购买时的价格为准

2. **产品可用性**
   - 部分产品可能因地区或时间原因暂时缺货
   - 可选择同类型其他产品替代

3. **安装要求**
   - 不同产品有不同的安装要求
   - 建议咨询专业安装人员

4. **个性化需求**
   - 如有特殊需求，请在"使用需求"中详细说明
   - AI会根据您的需求调整推荐方案

## 技术实现

### 知识库结构
```typescript
interface ACProduct {
  id: string;              // 产品ID
  brand: string;           // 品牌
  model: string;           // 型号
  type: string;            // 类型
  horsePower: number;      // 匹数
  suitableArea: {          // 适用面积
    min: number;
    max: number;
  };
  energyLevel: string;     // 能效等级
  price: {                 // 价格区间
    min: number;
    max: number;
  };
  features: string[];      // 产品特点
  bestFor: string[];       // 适用场景
  noise: number;           // 噪音
  cooling: number;         // 制冷量
  heating: number;         // 制热量
}
```

### AI集成方式
1. 将产品知识库转换为结构化文本
2. 注入到AI提示词中
3. 要求AI从知识库中选择产品
4. 确保推荐结果包含真实产品信息
